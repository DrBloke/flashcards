## Create decks for preparation for Exexcel Chemistry practicals exams

- [ ] Get a list of the key practicals from the specification
- [ ] Create a deck for each practical
- [ ] For each make sure to include:
  - [ ] Key skills, e.g. how to fill a burette
  - [ ] Calculations that might need to be performed
  - [ ] Sources of error in the experiment
  - [ ] Improvements
  - [ ] Diagrams
  - [ ] Safety
  - [ ] Common errors made by the student, according to mark schemes
